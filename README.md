# Developer Portfolio

Portfolio développeur de Jean-Christophe Parfait, construit avec React, TypeScript et Vite.

## Projets présentés

1. Tiny Act — application mobile Expo / React Native avec API Rails
2. Bill — bar conversationnel Rails avec LLM et API cocktail
3. Reelist — bibliothèque cinéma Rails avec TMDB et Pexels

## Développement local

```bash
npm install
npm run dev
```

## Vérifications

```bash
npm run typecheck
npm run build
```

Le site est déployé automatiquement sur GitHub Pages depuis la branche `main`.
